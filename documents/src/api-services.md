# API Routes - Services & Workflows

Documentation for services and workflow (AREA) management endpoints.

## Endpoints

### General
- GET /about.json
- GET /api/services

### Workflows (AREAs)
- POST /api/areas
- GET /api/areas
- GET /api/areas/:id
- PUT /api/areas/:id
- PATCH /api/areas/:id/toggle
- DELETE /api/areas/:id
- GET /api/areas/:id/logs
