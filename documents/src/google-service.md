# Google Services (Gmail & Drive)

## Overview

Google services integration provides automation for Gmail email monitoring and Google Drive file management.

## Services

### Gmail
- **Actions**: 2 (Email Received From, Email Subject Contains)
- **Reactions**: 0

### Google Drive
- **Actions**: 2 (New File in Folder, New File Uploaded)
- **Reactions**: 3 (Create File, Create Folder, Share File)

## Requirements

- Google account
- OAuth connection through AREA
