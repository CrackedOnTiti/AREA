# GitHub Service

## Overview

GitHub service enables automation based on repository events such as stars, issues, and pull requests.

## Features

- **Actions**: 3 (New Star on Repository, New Issue Created, New Pull Request Opened)
- **Reactions**: 1 (Create Issue)

## Requirements

- GitHub account
- Repository access
- OAuth connection through AREA
- Permissions: `public_repo`, `repo`
