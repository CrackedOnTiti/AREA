# Spotify Service

## Overview

Spotify service provides automation for music playback, playlist management, and library monitoring.

## Features

- **Actions**: 3 (Track Added to Playlist, Track Saved to Library, Playback Started)
- **Reactions**: 3 (Add Track to Playlist, Create Playlist, Start Playback)

## Requirements

- Spotify account (Free or Premium)
- OAuth connection through AREA
- Permissions: Playback control, playlist management, library access
